# RAG ChatBot

A Streamlit-based RAG chatbot for asking questions from websites and uploaded documents.

## Features

- Website URL based Q&A
- PDF, TXT, DOCX document Q&A
- Automatic document processing after upload
- FAISS vector database
- Ollama embeddings
- ChatGroq LLM
- Chat history in sidebar
- Page number display for documents

## Tech Stack

- Python
- Streamlit
- LangChain
- ChatGroq
- Ollama Embeddings
- FAISS
- PyPDF
- Docx2txt
